FRAPS 3.5.9 RELEASE NOTES

The system requirements for this version of Fraps are:
  • DirectX 9.0c, DirectX 10, or DirectX 11
  • CPU with SSE2 instructions (Pentium 4 and above)
  • 3D graphics card (Nvidia Geforce or ATI Radeon recommended)
  • Windows XP, Windows 2003, Windows Vista, or Windows 7 (including x64 editions)
    • This version of Fraps has been developed with the 301.42 Forceware drivers for NVIDIA Geforce cards, and the 12.6 Catalysts for ATI Radeon cards.  Please make sure your installed video card drivers are at least these versions!  Earlier drivers may suffer performance problems.

    For the latest information please visit the Fraps website at www.fraps.com
